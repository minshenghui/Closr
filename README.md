# Vue3 + Element Plus 项目

示例项目。

## 项目结构

```
vue3-element-plus-project/
├── src/
│   ├── main.js          # 入口文件，配置Element Plus
│   └── App.vue          # 主组件，包含Element Plus示例
├── index.html           # HTML模板
├── vite.config.js       # Vite配置
├── package.json         # 项目依赖
└── README.md           # 说明文件
```

## 安装依赖

推荐使用 pnpm（在这个项目中更稳定）：

```bash
pnpm install
```

或者使用其他包管理器：
```bash
npm install
# 或
yarn install
```

## 启动开发服务器

```bash
pnpm dev
# 或
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```
